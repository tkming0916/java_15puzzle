# 概要
大学の講義で作成した**15パズル**です．

開発言語はJavaを使用しました．

Javaで記述したネットワーク型対戦ゲームであれば何を作成しても良かったので，
715パズルを作りました．

# 機能など
- ネットワーク対戦型なので，2人でプレイが可能
- 4×4の「15パズル」と3×3の「8パズル」の2種類
- 「犬」「猫」「キリン」「数字」の4モード
- 上記の2種類×4モードの**合計8通りのパズル**でプレイ可能
- BGMのオンオフ
- リセットや自動判定などの基本的なゲームの処理
